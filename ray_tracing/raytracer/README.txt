一个实现漫反射和镜面反射的基本的光线追踪程序。
用codeblock13.12 编写，只需内置基本库即可。
当然不借助任何平台直接编译（以main.cpp为主程序）也是可以的。
Vector.h 与 Vector.cpp 实现了基本的空间矢量运算。
Sphere.h 与 Sphere.cpp 实现了球体类，球体与光源均用这个类表示
Colour.h 与 Colour.cpp 实现了颜色类。
光线追踪算法在主程序main.cpp中的trace函数里实现。
为简单起见，图片保存格式为ppm格式。可用ps或者任何一个ppm查看器查看。
场景添加可以直接在主程序中仿照示例进行，目前自定两个场景。
